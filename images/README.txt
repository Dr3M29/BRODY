Drop your 5 screenshots in this folder with these exact names, and the site
will show them automatically (until then, simple placeholders show instead):

  sketch.jpg        <- the pencil sketch of Brody
  wheel-front.png   <- the round wheel, front view with the dimensions
  wheel-inside.png  <- the internal frame / motor mount inside the wheel
  wheel-hub.png     <- the drive assembly sitting in the wheel
  wheel-side.png    <- the side profile with the small bracket on the left

Tip: names are case-sensitive on GitHub. .png or .jpg both work if you match
the name in index.html.
