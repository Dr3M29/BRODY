# Brody

A robot I'm building in Autodesk Inventor. Started as a sketch in my notebook.

**Milestone #1:** wheels + brackets modeled and assembled.

## The site
Open [`index.html`](index.html) in a browser for the milestone page (pictures +
short descriptions). The screenshots live in [`images/`](images/).

## What's in here
- `Brody.iam` / `Brodyv2.iam`: the main assembly
- `Wheel*.ipt`: the wheel (v1, v2, mirrors)
- `Wheel Bracket*.ipt`, `WheelBracketConnector.ipt`, `WheelBracketScrew*.ipt`: brackets + hardware
- `BrodyLegToWheelConnecter*.ipt`: leg-to-wheel connector
- `FastenerForLegFrame.ipt`: leg-frame fastener
- `OldVersions/`: earlier saves
